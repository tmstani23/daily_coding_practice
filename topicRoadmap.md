# Topic Roadmap

Work top to bottom; ~3-5 sessions per section depending on how it goes.
Mark sections done as they're completed. Every 3rd session, pull from
weakSpots.md instead of advancing.

## 1. JS core recall (start here — speed, not learning)
- Array methods: map/filter/reduce/find/some/every
- String manipulation, template literals
- Object/array destructuring, spread/rest
- Sorting with comparators

## 2. Data shaping (the daily-job skill)
- Grouping, counting, deduping with objects/Map/Set
- Nested data transforms (array of objects -> lookup, etc.)
- Chaining transforms cleanly

## 3. Async
- Promises from scratch (then/catch, chaining)
- async/await, error handling
- Promise.all / allSettled / race
- fetch patterns, sequential vs parallel

## 4. Closures, scope, functions
- Closure exercises (counters, once, memoize)
- this, bind/call/apply
- Debounce/throttle from scratch

## 5. TypeScript
- Typing functions, objects, generics basics
- Narrowing, unions, discriminated unions
- Utility types (Pick, Omit, Partial, Record)

## 6. Classic algorithms (interview staples, light touch)
- Two pointers, sliding window
- Frequency counters
- Basic recursion
- Binary search

## 7. Mini-project sessions (layer in 1-2x/week from section 3 onward)
- Small CLI tool or tiny API — decision-making practice, not drills
- Project TBD when we get there
